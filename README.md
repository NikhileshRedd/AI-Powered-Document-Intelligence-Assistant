# AI-Powered-Document-Intelligence-Assistant
Developed an interactive AI-based document assistant using Streamlit that enables users to upload PDF documents and ask natural language questions.

**AI-Powered Document Intelligence Assistant using RAG & Vector Search**
Developed an AI-powered document assistant web application using Streamlit for interactive PDF-based question answering.
Implemented PDF text extraction using pdfplumber to process and prepare unstructured documents.
Built a text chunking pipeline to segment large documents into smaller, manageable sections for efficient processing.
Generated semantic embeddings using Sentence Transformers (all-MiniLM-L6-v2) for contextual understanding of text.
Integrated FAISS vector database to perform fast and efficient similarity search over document embeddings.
Designed a retrieval-based QA system (RAG-style) to fetch the most relevant document sections based on user queries.
Created a responsive and user-friendly UI with Streamlit, including custom CSS for improved user experience.
Enabled real-time question answering from uploaded PDFs without relying on keyword-based search.
Improved document information retrieval accuracy using semantic search over traditional methods.
